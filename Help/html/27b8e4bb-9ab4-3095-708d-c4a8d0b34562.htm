ParseXsdBooleanValue(String)
Interprets the given string value as an xsd:boolean value. Returns true if the string value is "true" or "1", false if the string value is "false" or "0", or null if the string value is null. If the value is none of these, this method throws an exception.
ParseXsdBooleanValue(String, Boolean)
Interprets the given string value as an xsd:boolean value. Returns true if the string value is "true" or "1", false if the string value is "false" or "0", or the value defaultValue if the string value is null. If the value is none of these, this method throws an exception.
