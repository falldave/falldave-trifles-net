ParseXsdBooleanAttribute(XElement, XName)
Parses the xsd:boolean value of the indicated attribute, returning true for a value of "true" or "1", false for a value of "false" or "0", or null if the attribute does not exist. If the attribute exists but does not match any of the above values, an exception is thrown.
