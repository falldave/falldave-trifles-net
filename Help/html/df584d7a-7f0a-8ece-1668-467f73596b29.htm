FallbackT(IOptT, IOptT)
Returns (in a deferred fashion) an option which contains the value from this option, if any, or the value from the specified fallback option, otherwise. The value of source (and of option, if source is currently empty) is reevaluated every time the returned option's value is retrieved.
FallbackT(IOptT, FuncOptT)
Returns (in a deferred fashion) an option which contains the value from this option, if any, or another option containing the result of the specified function, otherwise. The specified function is not called unless source is empty. The value of source (and the result of getResult(), where applicable) is reevaluated every time the returned option's value is retrieved.
