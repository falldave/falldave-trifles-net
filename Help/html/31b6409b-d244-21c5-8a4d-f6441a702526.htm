ValueFixOptTKey, TValue
Returns an immediate, non-deferred option that contains the value of the dictionary at the given key, if the dictionary contains the key, or an empty option otherwise. Since this option is not deferred, its value reflects the state of the dictionary at the time this method is called and will not change.
ValueOptTKey, TValue
Returns a deferred option that will contain the value of the dictionary at the given key, if the dictionary contains the key, or an empty option otherwise. Since this option is deferred, repeated evaluations (of Fix, SingleT(IOptT), and so forth) reflect the current state of the dictionary rather than its state as of the call to this method.
