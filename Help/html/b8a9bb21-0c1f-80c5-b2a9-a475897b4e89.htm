Type: SystemByte
A byte array whose values from the index offset to the index offset - actualCount - 1, inclusive, where actualCount is the value returned by this method, are replaced by the bytes read.
offset
Type: SystemInt32
The offset in buffer at which to store the bytes read.
count
Type: SystemInt32
The maximum number of bytes to be read.

Return Value

Type: Int32
The total number of bytes read. This value is count if that many bytes are immediately available for read; otherwise, the value may be smaller. A value of 0 indicates that the end of the source has been reached.
